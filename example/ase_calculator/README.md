# CarNet ASE Calculator Example

This directory contains an example of using the CarNet ASE calculator.

## Files
- `run_ase.py`: The main script that loads a CarNet model and calculates properties for a structure.
- `liquid-64.xyz`: A sample XYZ file containing 64 water molecules.

## How to Run
Update `run_ase.py` to let `model_path` to your trained model checkpoint; then do 
`python run_ase.py`
